# Movie Slider
A simple movie slider app built with vanilla JavaScript. Users can navigate through movies using next and previous buttons, with smooth fade and slide animations.
### ![alt text](preview.png)
## Tech Stack & Features
- **HTML5** – Semantic markup for structure
- **CSS3** – Flexbox layout and transitions for animations
- **JavaScript (ES6+)**:
  - Modules (`type="module"`)
  - Arrow functions
  - Destructuring
  - Event listeners
  - DOM caching and manipulation
- **Responsive design** – Works across screen sizes
- **No external dependencies** – Pure JS implementation

## Features
- Smooth fade-in and slide-down animations for the entire movie card
- Looping slider: navigates from last to first movie seamlessly
- Dynamic content updates (image, title, category, info, year)
- Modular and maintainable JavaScript code structure
- Lightweight and performant due to DOM caching and minimal operations
- Modern ES6+ practices applied for cleaner, readable code
